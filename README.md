# 沖繩親子自駕行程

這是一個可部署到 GitHub Pages 的靜態旅遊行程網站。

## 網站內容

- 5/27 - 5/31 沖繩 5 天 4 夜行程
- 每日行程切換
- 每站 Google Map 與附近停車查詢
- 每日即時天氣預報
- 星宇航空航班狀態入口

## GitHub Pages 設定

1. 到 repository 的 `Settings`
2. 選擇 `Pages`
3. Source 選 `Deploy from a branch`
4. Branch 選 `main`
5. Folder 選 `/root`
6. 儲存後等待 GitHub 產生網址

網址通常會是：

```text
https://你的帳號.github.io/repository-name/
```
